Setting Up Bootstrap

Bootstrap 5 is linked via CDN in this page:

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">

Bootstrap JavaScript plugins are loaded with bootstrap.bundle.min.js at the end of the page.

Downloaded Bootstrap folder structure

  • css: compiled Bootstrap styles and utilities.
  • js: bundle for components like modals, collapse, dropdowns.
  • icons: optional Bootstrap Icons styles and fonts.
